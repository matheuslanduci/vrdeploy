# Estrutura

O projeto segue o padrão de monorepo, com a seguinte estrutura:

```
.
├── docs/                # Documentação do projeto
├── agent/               # Código do agente 
├── api/                 # Código do backend
├── shared/              # Código compartilhado entre os pacotes
└── ui/                  # Código da plataforma
```
